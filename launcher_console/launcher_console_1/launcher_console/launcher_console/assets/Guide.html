基本功能

主界面

CoogoOS的主界面由具有快捷功能的卡片组成，您可以左右滑动切换页面。

应用卡片:

ItemsPreviewsDescribe
天气显示当前位置的天气信息，您可以点击卡片更新天气信息。
导航当导航时，会显示导航信息。您可以点击图标启动一键导航。
音乐当有音乐播放时，会显示歌曲名称和专辑封面。您可以点击/图标播放/暂停音乐。点击/图标切换上一首/下一首歌曲。
行车记录当有行车记录仪连接时，会显示录像预览。当无行车记录仪连接时，会显示图标。您可以点击/图标开始/停止录像，点击/图标锁定/解锁当前录像，点击图标拍照。
收音机当收音机正在播放时，会显示当前收音频率。您可以点击/图标播放/停止收音机，点击/图标切换上一频道/下一频道，长按/图标搜索上一个/下一个可用频道。
蓝牙当设备通过蓝牙连接手机后，会显示所连手机的设备名称。当设备未连接手机时，会显示未连接。

应用抽屉

应用抽屉显示所有应用程序的图标，包括任何新安装的应用程序。

如要打开应用抽屉，请在主界面向下滑动，向上滑动可以返回主界面。

您可以左右滑动切换页面。

快捷设置

您可以通过快捷设置轻松改变常用设定，例如开启Wi-Fi。

如要开启快捷设置，请点击导航&状态栏中间的 图标。若要关闭，请再次点击。

如要变更设置，请点击相关图示:

  • 屏幕亮度 : 滑动即可降低或提高屏幕亮度。
  • 音量: 滑动即可降低或提高音量。
  • Wi-Fi网络: 点击开启/关闭Wi-Fi ，长按进入Wi-Fi设置界面。
  • 移动数据: 点击 开启/关闭移动数据，长按进入流量使用情况界面。
  • ADAS: 点击 开启/关闭ADAS播报。(需平台支持)
  • 屏幕休眠: 点击 进入屏幕休眠。

屏幕休眠

当您在夜间行车不想被屏幕打扰时，可以进入屏幕休眠。

如要进入屏幕休眠，请按以下步骤操作:

  1. 点击导航&状态栏中间的 图标。
  2. 点击 图标。

如要唤醒屏幕，请双击屏幕。

如要打开，请前往设置>WLAN热点。

如要调节，请点击音效 。

如要设置，请前往设置 >工厂设置 。

蓝牙

连接手机

要使用蓝牙电话及蓝牙音乐，请先连接手机:

  • 手机连接设备:

    1. 打开蓝牙应用 。
    2. 点击设置标签 ，打开蓝牙及设备可见性。
    3. 查看设备名称和PIN码。
    4. 在您的手机上打开蓝牙并搜索设备。
    5. 输入PIN码，连接设备。

  • 设备连接手机:

    1. 打开蓝牙应用 。
    2. 点击设置标签 ，打开蓝牙并扫描设备。
    3. 点击目标设备配对并连接。

拨打电话

拨打号码

要输入号码，请使用拨号键盘。

  1. 打开蓝牙应用 。

  2. 点击拨号标签 。

  3. 输入相应号码。

  4. 要拨打电话，请点击拨号图标 。

    • 删除: 如果您输错了号码，请点击删除图标 。
    • 结束通话: 点击结束通话图标 。

回电话

  1. 打开蓝牙应用 。
  2. 点击通话记录标签 。
  3. 此时，屏幕上会显示您的通话记录。您可以选择最近呼叫过的联系人或号码。

拨打联系人的电话

  1. 打开蓝牙应用 。
  2. 点击通讯录标签 。
  3. 要拨打电话，请从列表中选择一个联系人并点击。

通话过程中可用的选项

您可以在拨打电话时使用下列选项:

  • 拨号键盘: 要切换到拨号键盘，请点击 。
  • 声音通道: 要切换声音通道，请点击蓝色图标。 表示声音从设备发出， 表示声音从手机发出。

接听来电

当您接到电话时，屏幕上会显示含有来电者信息的通知。如果来电者是您通讯录中的联系人，则屏幕上会显示有关该来电者的详细信息。

  • 接听来电: 要接听来电，请点击接听 图标。
  • 忽略来电: 要忽略来电，请点击挂断 图标。

使用通话记录

您可以在通话记录标签中查看您的所有已拨电话、已接来电或未接来电列表。利用该功能，您可以方便地重拨、回拨。

要使用通话记录，请按照以下步骤操作:

  1. 打开蓝牙应用 。

  2. 点击通话记录标签。您将看到最近的来电和拨出的电话列表。

  3. 点击相应的按钮选择通话类型:

    • 未接来电
    • 已接来电
    • 拨出电话
    • 所有通话

  4. 对于最近通话记录，您可以执行以下操作:

    • 点击号码或联系人以回拨相应号码。
    • 点击清空清空通话记录。

管理联系人

当您通过蓝牙连接到手机时，可将手机中的联系人同步至设备。您可以在通讯录标签页中管理已同步的联系人。

对于联系人，您可以执行以下操作:

  • 点击搜索搜索联系人。
  • 点击清空清空联系人列表。
  • 点击联系人以拨打相应号码。

收音机

播放/停止

收音机播放时系统会将其它声道的音量调至最低。

您可以点击操作栏上的 /图标播放/停止收音机。

切换调频/调幅

您可以点击操作栏上的 图标切换调频/调幅。

调节频率

如要调节频率，您可以:

  • 点击频率调节区的 / 图标向上/向下调节频率。
  • 在频率数值区域向左/右滑动快速向上/向下调节频率。

搜索频道

收音机可以自动搜索可用频道。

如要搜索可用频道，您可以:

  • 点击操作栏上的 图标，收音机会自动搜索所有可用频道并储存至频道列表。
  • 长按频率调节区的 / 图标，收音机会自动搜索上一个/下一个可用频道并播放。

储存频道

频道列表一共有3页，可以储存18个频道。您可以在频道列表上左右滑动切换页面。

如要储存频道，您可以长按频道列表中任意一个频道位，当前收音频率会储存至相应的频道位。

选择频道

您可以在频道列表中点击想要播放的频道，该频道会变成绿色并播放。

胎压

如要使用胎压功能，请购买胎压配件。

添加设备

首次使用胎压，需要添加设备。

如要添加设备，请按照以下步骤操作:

  1. 打开胎压界面。
  2. 点击汽车图示周围对应轮胎的区域。
  3. 在弹出的对话框中输入设备ID, 设备ID可在胎压模块上找到。
  4. 点击确定。

添加设备后，如果连接设备成功，胎压数据和温度数据将会显示在汽车图示的周围。

变更设备

如要变更设备，请按照以下步骤操作:

  1. 打开胎压界面。
  2. 点击汽车图示周围对应轮胎的区域。
  3. 在弹出对话框中修改设备ID。
  4. 点击确定。

安装设备

如要了解如何安装胎压设备:

  • 参阅胎压设备自带的说明书.
  • 点击菜单图标 >帮助 。

警报

胎压警报

当胎压高于或低于所设压力值时，胎压数值将变红并闪烁，同时播放提示音。

如要设置胎压上下限值，请点击菜单图标 >设置 。

温度警报

当轮胎温度高于所设温度值时，温度数值将变红并闪烁，同时播放提示音。

如要设置温度上限值，请点击菜单图标 >设置 。

微信远程

您可以使用微信绑定设备，设备绑定成功后即可使用多种远程功能。

如要绑定设备，请按以下步骤操作:

  1. 在设备上前往设置 >设备绑定 。
  2. 点击二维码并用微信扫描弹出的二维码。
  3. 在手机上关注扫描二维码后显示的公众号。注: 如已关注该公众号，则无须重新关注。

设备管理

一个微信账号可以绑定多台设备，您可以在微信公众号中管理设备。

如要管理设备，请在公众号中点击更多>设备管理，然后点击收到的消息进入设备管理页面。

设置当前设备

一个微信账号只能同时与一台设备交互，您可以在设备管理页面设置当前设备。

如要设置当前设备，请在设备管理页面中选定相应设备的设备号。

如要查看设备号，请在设备上前往设置 >设备绑定 >IMEI号。

解绑设备

一台设备只能被一个微信账号绑定，如要更换微信账号，请先使用原微信账号解绑设备。

如要解绑设备，请在设备管理页面中选定相应设备的设备号，并点击解除绑定。

位置查询

您可以通过微信查看设备的位置。如要查看设备位置，请在微信公众号中点击我的设备，然后点击收到消息中的位置查询打开位置查询页面，设备位置会在地图上显示。

轨迹回放

您可以通过微信查看安装有设备的车辆的行车轨迹。

如要查看行车轨迹，请按以下步骤操作:

  1. 在微信公众号中点击我的设备。
  2. 点击收到消息中的轨迹回放打开轨迹回放页面。
  3. 选择日期并点击确定，该天的行车轨迹会在地图上显示。

好友指路

您可以向微信联系人发送好友指路请求，对方收到请求并同意后，设备会收到该联系人的地理位置并开启导航。

如要使用好友指路，请按以下步骤操作:

  1. 在微信公众号中点击我的设备。
  2. 点击收到消息中的好友指路打开好友指路页面。
  3. 点击页面右上角的菜单按钮。
  4. 点击发送给朋友。
  5. 选择微信联系人并点击确定。
  6. 点击发送。

联系人收到好友指路请求并同意后，设备会弹出好友指路提示，点击确定即可导航至该联系人的所在位置。

位置共享

您可以将设备的位置共享给您的微信联系人。

如要使用位置共享，请按以下步骤操作:

  1. 在微信公众号中点击我的设备。
  2. 点击收到消息中的位置共享打开位置共享页面。
  3. 点击页面右上角的菜单按钮。
  4. 点击发送给朋友。
  5. 选择微信联系人并点击确定。
  6. 点击发送。

联系人收到位置共享页面后，可在两小时内查看设备的实时位置。

预约导航

您可以通过微信预先设定好设备的导航目的地，上车即可导航。

如要使用预约导航，请按以下步骤操作:

  1. 在微信公众号中点击我的设备。
  2. 点击收到消息中的预约导航打开预约导航页面。
  3. 在页面顶部的搜索框中输入目的地。
  4. 点击启动预约导航。

设备在收到预约导航指令后，会弹出预约导航提示，点击确定即可导航至设定的目的地。

电子围栏

您可以通过微信在地图上设定电子围栏，当设备超出围栏范围时，微信将会收到提醒。

如要开启电子围栏，请按以下步骤操作:

  1. 在微信公众号中点击我的设备。
  2. 点击收到消息中的电子围栏打开电子围栏页面。
  3. 拖动地图选择电子围栏中心点。
  4. 点击开启电子围栏。
  5. 选择电子围栏半径。
  6. 点击启动电子围栏。

如要关闭电子围栏，请按以下步骤操作:

  1. 在微信公众号中点击我的设备。
  2. 点击收到消息中的电子围栏打开电子围栏页面。
  3. 点击关闭电子围栏。

音乐推送

您可以通过微信指定要在设备上播放的音乐。

如要使用音乐推送，请按以下步骤操作:

  1. 在微信公众号中点击我的设备。
  2. 点击收到消息中的音乐推送打开音乐推送页面。
  3. 在搜索框中输入歌曲名并点击搜索。
  4. 选择想要推送的歌曲并点击推送按钮。

远程关机

如果您长时间不使用车辆，为防止车辆电瓶被耗尽，您可以通过微信远程关闭设备。

如要关闭设备，请在微信公众号中点击更多>远程关机。

注：设备关闭后将无法使用微信远程功能

远程语音

您可以通过向微信公众号发送语音消息控制设备。

如要使用远程语音，请按以下步骤操作:

  1. 在微信公众号中点击 切换到输入模式。 如已切换到输入模式，请跳过此步骤
  2. 点击切换到语音输入模式。 如已切换到语音输入模式，请跳过此步骤
  3. 长按按住说话并说出语音指令。
  4. 松开手指发送指令。

违章查询

您可以通过微信查询违章信息。

如要使用违章查询，请按以下步骤操作:

  1. 在微信公众号中点击更多。
  2. 点击违章查询打开违章查询页面。
  3. 输入车牌号码、车架号码和发动机号等信息。
  4. 点击开始查询。
